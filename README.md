# career-echo-openworld-mvp

MVP רץ בדפדפן של משחק חקר קריירה תלת־ממדי בעולם פתוח (גוף ראשון), עם:
- ממלכת "יער" אחת (מתוך 3 מתוכננות)
- 3 סצנות אינטראקטיביות (מתוך 10 מתוכננות)
- בחירות שמעדכנות פרופיל RIASEC + HUD
- דיאלוגים + Echo קצר
- יומן רפלקטיבי + תיק פריטים
- שמירה/טעינה מקומית ב־localStorage
- טלמטריה מקומית + ייצוא לקובץ JSON
- שליטה במחשב ובמובייל

## דרישות

דפדפן מודרני עם WebGL:
- Chrome / Edge / Firefox / Safari עדכניים
